Markdown简明语法

发表于 2016-01-04 | 阅读次数

1. 粗体和斜体

使用 * 和 * 表示斜体和粗体。
示例： 斜体 `斜体**粗体***粗体`

2. 分级标题

在文本前加#即表示标题，共六级，字号依次减小
示例：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

3. 列表

<1>无序列表

无序列表只需在文本前加上-
举例：

  • 第一章
  • 第二章
  • 第三章

<2>有序列表

有序列表需在文本前加上1. 2.
举例：

  1. 第一章
  2. 第二章
  3. 第三章

4. 链接和图片

在Markdown中，插入链接的格式为[文本] (链接地址)
举例：

虎扑体育
在Markdown中，插入图片的格式为![] (图片的链接)
举例：

5. 引用

在Markdown中，引用只需在文本前加上>即可
注： > 和文本之间需要添加一个空格
举例：

披绣闼，俯雕甍，山原旷其盈视，川泽纡其骇瞩。闾阎扑地，钟鸣鼎食之家；舸舰迷津，青雀黄龙之舳。云销雨霁，彩彻区明。落霞与孤鹜齐飞，秋水共长天一色。渔舟唱晚，响穷彭蠡之滨；雁阵惊寒，声断衡阳之浦。

6. 代码引用

  1. 行间引用举例：

行间引用

  1. 多行引用举例：
    多行引用只需在头尾各添加```即可，头部还需加上编程语言名称
# Markdown
