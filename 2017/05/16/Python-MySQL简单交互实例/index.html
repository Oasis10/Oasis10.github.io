二、connect对象

  • connect对象用于与MySQL数据库建立连接
  • 使用connect方法创建对象

  • 参数列表

    host:  连接的MySQL主机，如果是本机则为”localhost”
    port:  连接的MySQL主机端口，默认3306
    user:  连接的MySQL主机用户名
    passwd:  连接的MySQL主机密码
    db:  连接的MySQL主机某个数据库
    charset:  通信编码方式，建议”utf8”

  • 对象方法

    close() 关闭连接
    commit() 事务，所以需要提交才会生效
    rollback() 事务，放弃之前的操作
    cursor() 返回cursor对象，用于执行sql语句并获得结果
